## 🌟 About the Project

**FoodLink** is a socially impactful web application dedicated to **reducing food waste and feeding those in need** by connecting local messes and food providers with orphanages and old age homes.

🍛 Donors can post available surplus meals, while NGOs and institutions can browse and request the food — ensuring it reaches the right hands at the right time.  
🌍 A small initiative to promote sustainability, social equity, and community-driven support.

🔗 **Live Site:** _Coming Soon – Hosted on GitHub Pages_

---

## 🔥 Key Features

- 👤 **User Roles:** Separate portals for Donors & Receivers
- 📦 **Food Listings:** Post, update, and track food availability
- 📍 **Live Location (Coming Soon):** Efficient logistics for pickups
- 📊 **Admin Dashboard:** View metrics and manage users
- 🔔 **Notification System (Planned):** Alerts for nearby availability
